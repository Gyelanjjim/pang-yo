## 🏆 PangYo 목표

- 제빵사와 빵 컨셉의 게시판 구현
- 사용자의 게시글 투표에 따라 게시글(`빵`) 색에 변화를 주어 잘 구운 빵과 탄 빵의 컨셉을 구현
- AWS EC2, Docker를 사용한 웹사이트 배포
